# HBM's Nuclear Tech Mod: Final Edition

> [!NOTE]
> NTM FE is a solo passion project. It tracks NTM: Community Edition upstream and stays broadly compatible with its direction, but it is not an official CE fork, not affiliated with the CE team, and not intended as a replacement for it. If you want the mainline experience, go play CE. If you want unhinged reactor coolant options and features CE would (rightly) never ship, you're in the right place.

> [!WARNING]
> NTM FE is currently a development fork. There are no official builds yet because apparently "making sure things don't explode in unexpected ways" is something that takes time.
> 
>Expect unfinished features, balance changes, and things to randomly change because I felt like it.

## What is this?

NTM FE is literally just NTM: Community Edition, except I take the parts I like and then make questionable decisions with them.

Some changes aim for more interesting or realistic mechanics. Some exist because I thought it would be funny.

For example, you can now cool an RBMK with literal antimatter.

Should you?

*No*.

Can you?

*Absolutely*.

Is that the important part?

**Obviously**.

The general rule is:
I do whatever the hell i want.

## Notable additions over base NTM

### RBMK Universal Coolant Expansion

The RBMK originally had a rather small selection of valid coolants.

This has been corrected by doing the reasonable thing: allowing basically everything.

By "reasonable", I mean:

- Mercury
- Hydrogen
- Tritium
- Nitroglycerin
- Antimatter
- Mustard Gas
- Nuclear Waste
- Estradiol Solution
- Sunflower Oil

You know. Normal reactor coolant choices.

### Type I Compliance Module

Added the Type I Compliance Module.

It is an endgame power block that burns an entire dimension for fuel and just *barely* makes enough energy to classify you as a Type-I Civilization.

This is probably fine.

(It's not.)

### Nuclear Waste RTG Pellets

Added Nuclear Waste RTG Pellets.

Because radioactive waste wasn't already doing enough.

### Other changes

- The BAT 9000 can now store antimatter.
- Added microwave sounds.
- Added white, yellow, and red steel scaffold variants.
- Added paperclips. (yes)

More additions will be documented as development continues.

## Current state

The mod is not finished.

There is no roadmap.

There is a vague list of things I want to do, and I will probably work on something completely different instead.

Features may appear when they appear.

## Is it survival ready?

Mostly, with the understanding that this is a personal fork under active, sometimes chaotic development, expect things to be in flux more often than a stable release branch would be.

## Compatibility

Not guaranteed to be compatible with NTM: Extended Edition addons, other NTM forks, or worlds created on those forks. Treat this as its own thing, not a drop-in replacement for CE or any other fork.

## Development

Built on top of NTM: Community Edition.
